## Overview

ShaderFoundry is a browser-based GLSL shader playground. Write fragment shaders
in a Monaco-powered editor, see results in real time on a WebGL canvas, and share
your creations via URL.

## Tech Stack

- **TypeScript** — Application logic
- **WebGL 2** — Shader rendering
- **Monaco Editor** — Code editing with GLSL syntax highlighting
- **Vite** — Build tooling

## Key Features

- Live preview with configurable uniforms (time, resolution, mouse)
- Built-in shader gallery with community contributions
- Shareable URLs with encoded shader source
- Fullscreen rendering mode

## Challenges

Handling shader compilation errors gracefully required parsing the WebGL error
log and mapping error lines back to the editor, since GLSL error messages
reference the combined source (including injected preamble lines).

## What's Next

This project is complete for now. Potential future additions include multi-pass
rendering and audio-reactive uniforms.
