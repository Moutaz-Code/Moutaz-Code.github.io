## Overview

HexaSentinal is an offline digital forensics assistant designed for investigators
and security researchers. It features modular ingestion pipelines that can parse
disk images, memory dumps, and log files, feeding structured data into an
AI-powered analysis engine.

## Tech Stack

- **.NET 8** — Core runtime and CLI tooling
- **WPF** — Desktop UI with dockable panels and timeline views
- **SQLite** — Local evidence database
- **ML.NET** — On-device anomaly scoring

## Key Features

- Modular evidence ingestion (plug-in architecture for new formats)
- Timeline reconstruction from filesystem metadata
- AI-assisted anomaly flagging with explainability scores
- Fully offline — no cloud dependencies

## Challenges

Building a responsive UI that remains snappy while processing large disk images
required careful background task management and virtualized list rendering in WPF.
